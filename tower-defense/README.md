# Cursor Crew — Tower defense (Defrag Run)

Three.js arena prototype: protect the **Recycle Bin**, fill the **upload** meter, survive waves. Active development lives here; the jam Canvas game is in [`../shooting-game/`](../shooting-game/).

## Develop

```bash
npm install
npm run dev
npm test
npm run build
```

## E2E (optional)

```bash
npx playwright install
npm run verify
```

## Jam widget

[`index.html`](index.html) includes the Vibe Jam 2026 widget script when submitting builds that need it.
